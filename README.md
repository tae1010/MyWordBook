# MyWordBook 
### 주제 : ocr을 이용한 단어장 어플리케이션<br>
<hr>

## 프로젝트 소개
학생들이 공부하는 시간에 단어장에 단어를 하나하나 입력하기가 불편하고 시간이 많이 소모되기 때문에 ocr을 이용해 자동으로 단어를 입력해주는 어플리케이션
<br>
<hr>

## 팀 구성<br>
* 김정태<br>
* 곽재선<br>
* 김동현<br>
* 심정수<br>
<hr>
